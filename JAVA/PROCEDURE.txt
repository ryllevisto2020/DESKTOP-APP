4.connect SSH using .connect
5.Create Channel Exec using .openChannel("exec") or Create Channel Shell using .openChannel("shell")
6.Set Command in Channel Exec using .setCommand(<Type your Command>)
7.Create Response using ByteArrayOutputStream

10. Disconnect Channel Exec using .disconnect
